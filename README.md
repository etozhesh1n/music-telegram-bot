# 🎵 Music Telegram Bot

Telegram бот для скачивания музыки с YouTube и отправки в Telegram.

## Возможности

- 🔍 Поиск треков на YouTube
- ⬇️ Скачивание в MP3 формате
- 📤 Автоматическая отправка с метаданными (исполнитель, название)
- 🧹 Автоматическая очистка приписок типа "(Official Audio)", "[Lyrics]"
- 🌐 Поддержка русского и английского языков

## Использование

Отправьте боту название трека в любом формате:
- `Исполнитель - Название`
- `Название трека`

Бот найдет трек на YouTube, скачает и отправит вам.

## Команды

- `/start` - Приветствие
- `/help` - Справка

## Деплой на Render

1. Форкните этот репозиторий
2. Зарегистрируйтесь на [Render.com](https://render.com)
3. Создайте новый Web Service
4. Подключите ваш GitHub репозиторий
5. Добавьте переменные окружения:
   - `BOT_TOKEN` - токен вашего Telegram бота (от @BotFather)
   - `CHAT_ID` - ваш Telegram ID (от @userinfobot)
6. Deploy!

## Требования

- Python 3.11+
- FFmpeg
- Telegram Bot Token
